RAG CHATBOT BACKEND PLAN (LANGGRAPH MINIMAL ARCHITECTURE)

ARCHITECTURE GOAL
-----------------
Build a scalable LangGraph-based RAG backend with:
- FastAPI
- PostgreSQL
- pgvector
- LangGraph orchestration
- Retrieval node
- Generation node
- Streaming-ready architecture

Graph Flow:
START
  ↓
retrieval_node
  ↓
generation_node
  ↓
END
